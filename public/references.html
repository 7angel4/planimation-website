If you would like to explore more about PDDL and planimation, check out the resources below!

  • PDDL Editor

    The online editor where you can install the planimation plugin, then model and visualise your animations via planimation.

  • An introduction to the planning domain definition language

    A book which presents a unified and current account of PDDL, and introduces readers to the art of modelling planning problems in this language. Click on the link "access via your institution" to access the book through your university library.

  • Planimation Visualiser

    A visualiser that accepts a domain file, a problem file and an animation file. An animation with an user interface is produced based on the supplied files.

  • Planimation Documentation on GitHub

    Definitive home of Planimation. This site contains extended descriptions regarding the development of Planimation.

  • Planning.Domains

    This site contains a collection of tools for working with planning domains, including an API that provides programmatic access to a wide collection of PDDL benchmark domain and problem files, an automated planner and validator in the cloud, a PDDL editor, and a collection of learning resources.

  • Planning.Wiki

    Definitive home of PDDL, which focuses on PDDL syntax and interpretation of PDDL problems.
